Head

Head adalah elemen html yang digunakan untuk menyimpan metadata file html tersebut.

Metadata yang disimpan dapat berupa character encoding, setting viewport, judul file, dan lain-lain.

Elemen head tidak akan muncul secara langsung pada browser, namun elemen ini menyampaikan

informasi dan instruksi kepada browser atau software yang memproses file html tersebut.
